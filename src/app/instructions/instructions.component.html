A Cooperative Tuning Game

Overview

Us vs. It is a cooperative game that teaches iterative tuning. The game is played in two stages. In the first stage, you act as scientists working together to program a giant robot using plain text triggers to allocate actions. In the second stage, you take the role of tank commanders and control the game using the button controls on the game screen.

Setup

  1. Step 1: Select the "Programming" tab. As scientists, create a program for the robot using plain text triggers to allocate actions. Use your creativity to design actions and triggers that make the robot strong enough to progress across the board without crossing the goal line.
  2. Step 2: Select the "Game" tab. Once you've finished programming the robot, the game commences. Control the tanks using the button controls provided on the game screen. Coordinate your attacks to stop the robot before it reaches the nearby city.

Objective

The objective is to create a robot that is powerful but not too powerful. Aim for a dramatic and exciting game rather than an easy victory. Keep fine-tuning the robot's program and battling it until you find the perfect combination of actions that make for a thrilling Hollywood-style ending.

Pro Tips

Remember, you can program "flaws" into your robot to weaken it and give it personality. Be imaginative and have fun as you work together to create the ultimate robot and save the city!
